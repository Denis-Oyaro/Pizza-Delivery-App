# This file contains instructions on how the client can interact with the API

# Note:
1. Entry file of app is index.js
2. Provide the following environment variables inorder for mailgun integration to work:
      a) NODE_MAILGUN_API_KEY (this is the api key from mailgun)
      b) NODE_MAILGUN_DOMAIN_NAME (this is the sandbox domain name provided by mailgun)

#  About the API
I have included a postman collection in the zip file (pizza-delivery-app.postman_collection.json). Please import it into postman if you want to test the API.

# Access the Web GUI
The root of the server ('/') is the landing page for the web GUI.
